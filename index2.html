TIC TAC TOE FOREVER!

How to play

There are 2 players, player X and player O. Player X goes first. X clicks on an empty space in the grid to place an X in it. Then player O clicks an empty space in the grid to place an O in it. The first player to fill in 3 spaces in a row, vertically, horizontally or diagonally wins! If neither player fills in 3 spaces in a row, the game is a draw and both players start again.
